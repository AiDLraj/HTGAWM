# HTGAWM

An NLP Project to analyze the textual data of Court cases.

The court cases includes:

- Trayvon Martin
- Philando Castile
- Walter Scott
- Terrence Crutcher
- Tamir Rice
- Sandra Bland
- Samuel Dubose
- Oscar Grant
- Michael Lorenzo Dean
- Eric Garner
- Christopher Alexander Okamoto
- Chinedu Valentine Okobi
- Botham Shem Jean
- Atatiana Jefferson
- Antwon Michael Rose II
